# 🎁 Happy Anniversary — เว็บวันครบรอบ

เว็บกล่องของขวัญเปิดเซอร์ไพรส์ ธีมชมพูพาสเทลโรแมนติก ภาษาอังกฤษทั้งหมด
ครบรอบ **29 มีนาคม** · นับวันแบบเรียลไทม์

## วิธีเปิดดู
ดับเบิลคลิกไฟล์ `index.html` เพื่อเปิดในเบราว์เซอร์ได้เลย
(หรือเปิดผ่าน Live Server ใน VS Code ก็ได้)

## สิ่งที่อยู่ในเว็บ
1. หน้าแรก — กล่องของขวัญเรืองแสง กดเพื่อเปิด ✨
2. หลังเปิด — เผยข้อความครบรอบ พร้อม:
   - ⏳ ตัวนับวัน/ชั่วโมง/นาที/วินาที ที่อยู่ด้วยกัน (เริ่มนับ 29 มี.ค. 2026)
   - 🖼️ สไลด์รูปภาพ
   - 💌 จดหมายรัก
   - 🎵 เพลงประกอบ

## วิธีปรับแต่งให้เป็นของเราจริงๆ

### 1) เปลี่ยนข้อความ / จดหมาย
แก้ใน `index.html`
- ข้อความหัว ("Three Months With You", "Happy Anniversary, my love")
- เนื้อจดหมายในส่วน `<div class="letter-body">`
- คำลงท้าย `class="signature"`

### 2) ใส่รูปภาพ
วางไฟล์รูปในโฟลเดอร์ `images/` ตั้งชื่อ `photo1.jpg`, `photo2.jpg` ...
(ดูรายละเอียดใน `images/README.txt`)

### 3) ใส่เพลง
วางไฟล์เพลงในโฟลเดอร์ `music/` ตั้งชื่อ `song.mp3`
(ดูรายละเอียดใน `music/README.txt`)

### 4) เปลี่ยนวันที่เริ่มต้น (ตัวนับวัน)
แก้ใน `js/main.js` บรรทัด:
```js
const START_DATE = new Date(2026, 2, 29, 0, 0, 0); // เดือนนับจาก 0 → 2 = มีนาคม
```

## โครงสร้างไฟล์
```
Nampan/
├─ index.html        ← หน้าเว็บหลัก + ข้อความทั้งหมด
├─ css/style.css     ← ธีม/สไตล์
├─ js/main.js        ← การทำงาน (วันที่, สไลด์, เพลง)
├─ images/           ← ใส่รูปที่นี่
└─ music/            ← ใส่เพลงที่นี่
```
